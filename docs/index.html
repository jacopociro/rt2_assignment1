User Interface Module¶

This node implements an user interface

Service:

/user_interface

scripts.user_interface.main()[source]¶

This function initializes the ROS node and waits for the user to insert start or stop to control the robot, by relying on the rospy module.

The user message is passed to the service user_interface, advertised by go_to_point.

GoToPoint Module¶

ROS node for driving a robot to a specific point

Subscribes to:

/odom topic where the simulator publishes the robot position

Publishes to:

/cmd_vel the desired robot position

Service :

/go_to_point to start the robot motion.

Position Service¶

This files implements a position service.

Services :

° /position_server

Author

Carmine Recchiuto

Version

0.1

Date

27/05/2021

Description :

This node advertises a position service. When the service a required, a request containining minimum and a maximum values for the x and y position is used to generate a random position between x (or y) min and x (or y) max.
